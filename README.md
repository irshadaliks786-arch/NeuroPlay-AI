# 🧠 NeuroPlay AI

> Explainable Machine Learning for Gaming Addiction Risk Assessment

## 📌 Overview

NeuroPlay AI is an end-to-end Explainable AI (XAI) project that predicts Gaming Addiction Risk using behavioral and psychological features.

Unlike traditional black-box models, NeuroPlay AI explains every prediction using SHAP values and provides personalized recommendations along with a What-If Simulator to estimate how lifestyle changes can affect addiction risk.

## 🧠 Explainable AI

SHAP is used to explain

- Global Feature Importance
- Local Prediction Explanation
- Personalized Recommendations

---

## 🔮 Prediction Engine

The model predicts

- Addiction Score (0–10)
- Risk Category
- Confidence Interval

---

## 💡 Recommendation Engine

Provides personalized recommendations based on SHAP feature importance.

Example:

- Reduce daily gaming hours
- Improve sleep schedule
- Increase physical activity
- Reduce stress levels

---

## 🔄 What-If Simulator

Users can simulate lifestyle changes and instantly observe how their predicted addiction score changes.

Example:

Current Score → 8.42

↓

After Lifestyle Changes

↓

Predicted Score → 7.92
